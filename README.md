## House Price Prediction

Explored the classic house price dataset to understand regression fundamentals.

- Model: Linear Regression with feature scaling
- Focus: Data exploration, preprocessing, and model evaluation
- Key learning: How feature scaling affects model performance


## Spam Classifier

Built a text classification model to distinguish spam from legitimate messages.

- Model: Naive Bayes on TF-IDF features
- Dataset: SMS Spam Collection
- Key learning: Text vectorization and working with imbalanced classes
